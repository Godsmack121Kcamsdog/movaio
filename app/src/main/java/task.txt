Задание: позволить пользователю искать фотографии по введенному слову, с помощью API.
Сохранять историю поиска: запрос и первую найденную фотографию.

Интерфейс: Сверху экрана - поле для поиска.

Ниже - список истории поиска (картинка-результат, и слово, которое искал пользователь).

Логика: после введения слова в строку поиска (например, "cat"),
и нажатия на Done/Enter на клавиатуре – начать поиск картинки.

Для поиска используется API от GettyImages.
Получив ответ от сервера, если она имеет картинку – нужно сохранить ее в истории поиска,
и отобразить ее в списке ниже.
Если фотография не была найдена – уведомить пользователя.

В качестве БД нужно использовать Realm.
